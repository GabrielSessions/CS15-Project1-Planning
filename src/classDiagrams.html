The passenger class models a passenger on a train and can be inserted into a queue or a train to simulate the metro. Passenger objects contain information to let the program know information about the person’s trip on the metro.

+ print(std::ostream &output)

A PassengerQueue object models a line of passengers waiting at a station to get onto a train. The PassengerQueue allows the program to add passengers at the back and remove them from the front when moving them to the train. Note: Passenger is returned when dequeuing so the person can be added to the train.

Edge Cases

  • Ensuring the train moves from the last station to the first at the end of the line
  • Not removing Passengers from the train when it’s empty or not moving Passengers to the train when the station is empty (avoid empty list errors)
  • File not found/able to load when starting the program
  • Printing the queue when there’s no Passengers or the list when there’s no passengers on the train
  • Invalid user input with std::cin (do nothing)
  • Handling cases where the arrival and departure station are the same or where the arrival station is greater than the departure station.
