CS 15 Project 1: Metro Sim Planning

Project Overview

A brief overview of the major components of this project. More detailed descriptions can be found later on this page. File names are subject to change.

  • main.cpp - Driver File
  • Makefile - Easily allows us to create an executable to run the sim
  • MetroSim.cpp - Implementation of the Metro System (train/passenger interation)
  • MetroSim.h - Interface for the Metro System
  • output.txt - Logs passengers that leave the train
  • Passenger Class (.h and .cpp) - Class definition of a passenger
  • PassengerQueue Class (.h and .cpp) - Class definition of a queue that holds Passenger objects
  • README - Lists general info about the files in this project and the project as a whole
  • stations.txt - List of stations on the Green Line (E)
  • text_commands.txt - List of commands fed into the simulation (through main.cpp)
  • the_MetroSim - A demo to compare my program to
  • Train Class (.h and .cpp) - Allows to control the movement of the train and passenger flow
  • unit_test.h - A place to write tests
